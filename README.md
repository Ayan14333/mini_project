# mini_project-(Bella Vista)

## Project Description
A responsive Italian Resturant Landing Page built using HTML, CSS, and Flexbox, enhanced with a custom CSS animation for the hero section image.

## Features
- Responsive Navigation Bar
- Flexbox-based layout
- Hero section with text + image
- Book a Service button 
- Mobile responsive design
- Custom CSS animation (no JS)

##  How to run it 
1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html** or open it in any web browser.
